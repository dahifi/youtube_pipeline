# YouTube Pipeline - yt-dlp Cheat Sheet

## 🎧 Extract Audio Only (MP3)
Extracts audio from a YouTube video and saves it as an MP3 file.

```bash
YID="<your_video_id>"
yt-dlp -x --audio-format mp3 --audio-quality 0 -o "outputs/%(title)s.%(ext)s" "https://www.youtube.com/watch?v=$YID"
```

## 📝 Download Transcript Only (Subtitles)
Downloads auto-generated or manually uploaded subtitles (if available) without downloading video/audio.

```bash
YID="zPmCPkzfosI"
yt-dlp --write-auto-sub --sub-lang en --skip-download -o "outputs/%(title)s.%(ext)s" "https://www.youtube.com/watch?v=$YID"
```


- `--write-auto-sub`: gets YouTube's autogenerated subtitles
- `--skip-download`: no media, just subtitles
- Use `--write-sub` instead of `--write-auto-sub` for official subs
- `--convert-subs=srt`: converts subtitles to `.srt` which is more readable than `.vtt` and easier to post-process.
- Optional post-step: use a script or `ffmpeg` to strip timestamps if plain text is desired.

## 📹 Download Video and Remux to MKV
If video is needed in `.mkv` format:

```bash
YID="g9eFUFMznTA"
yt-dlp --remux-video mkv -o "outputs/%(title)s.%(ext)s" "https://www.youtube.com/watch?v=$YID"
```

---

Scripts to automate these tasks will live in a `/scripts` folder.
